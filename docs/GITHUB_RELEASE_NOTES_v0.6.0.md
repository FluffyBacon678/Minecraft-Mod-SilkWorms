# Silkworms v0.6.0 — Survival Beta 🐛🦋

First beta release! The full loop is in and survival-ready:

**Find** silkworm colonies on the moss in lush caves (or meadows-edge
forests) → **raise** them on grass and leaves → watch them spin **hanging
cocoons** → hatch elegant **silk moths** → **tame** one with cherry leaves →
**fly it**.

## Highlights

- 🐛 Complete lifecycle with persistent timers and 8 natural worm colours
  (cocoons inherit the tint)
- 🪣 Silkworm Bucket — carries 5 worms and remembers each one's colour
- 🦋 Refined silk moth (broad swept wings, layered eye-spots) is the one and
  only adult look
- 🤝 Tamed companion: 2× size, harness, immortal, better-than-Allay follow,
  wolf-style defense (hostiles only, never creepers)
- 🕹️ Owner-only riding: look-steered flight with anti-dive assist, jump to
  climb, hover, safe sneak-dismount — ~2× Happy Ghast speed
- 🕳️ Lush-cave natural spawning (packs of 2–5, never in water)
- ⚙️ Mod Menu config + phase-out toggles
- 🧹 Honest safe-removal commands (tamed moths become named vanilla Allays)

## Requirements

Minecraft **1.21.11** · Fabric Loader ≥ 0.16.0 · **Fabric API** · Java 21
· Mod Menu optional · required on both client & server in multiplayer

## Compatibility

No mixins, no access wideners, no custom networking, no blocks/world data.
Verified in a 119-mod pack (Sodium/Iris/Lithium). Removal guide:
`COMPATIBILITY_AND_REMOVAL.md`.
